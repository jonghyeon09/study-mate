## 프로젝트 소개

스터디 인증 목적으로 많이 사용하는 플랫폼으로 네이버 밴드, 카카오톡 오픈채팅 등이 있습니다.<br />
하지만 스터디 전용 서비스가 아니다 보니 불편한 점이 있어서 스터디 전용 서비스를 기획하게 되었습니다.<br />
현재 모바일 사이트로 제작되어 있습니다.

## 기능

- 카카오톡 로그인
- 스터디 생성, 수정, 종료
- 스터디 초대
- 스터디 인증(사진 업로드)
- 인증 글 확인(무한 스크롤)
- 공지사항 작성
